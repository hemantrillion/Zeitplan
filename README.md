# ZEITPLAN
### Schedule & Attendance Tracking App for Android

**Version:** 1.0.0 — Released 30 April 2026
**Platform:** Android only
**License:** Apache 2.0

---

## What is ZEITPLAN?

ZEITPLAN is a personal timetable and attendance tracking Android app.
It replicates a full semester schedule and lets you mark attendance
for each class directly from your phone — with color coding,
auto-save, and per-subject statistics.

---

## Features

- Full semester timetable (74 days, 7 time slots per day)
- Tap any subject to mark attendance instantly
- Color coded status:
  - Green = Present
  - Red = Absent
  - Orange = Medical Submitted
  - Dark Green = Medical Uploaded
  - Gray = No Class
- Lab cells (TA LAB, AC LAB) fully supported as merged 2-hour slots
- Attendance % summary per subject with 75% threshold indicator
- Auto-saves all changes to phone storage — nothing is lost
- Opens automatically at your last filled row on every launch
- Dark and Light theme support (Midnight Dark / Bright Light)
- Password protected — requires access key to open
- Clean minimal UI with ZEITPLAN branding

---

## How to Use

- Tap any subject cell → select attendance status from popup
- Tap Stats → to view per-subject attendance breakdown
- Tap the ZEITPLAN logo 3 times quickly → opens Settings
- In Settings you can switch between Dark and Light theme

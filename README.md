# ZEITPLAN

### Schedule & Attendance Tracking App for Android

**Platform:** Android only
**License:** Apache 2.0

---

## What is ZEITPLAN?

ZEITPLAN is a personal timetable and attendance tracking Android app.
It allows users to create their own schedule and mark attendance for each class directly from their phone — with color coding, auto-save, and per-subject statistics.

---

## Features

1. Create fully custom timetables

2. Select working days

3. Define your own time slots for each day

4. Multiple timetables supported (no overlapping dates allowed)

5. Tap any slot to update attendance instantly

6. Status Types

   * PRESENT
   * ABSENT
   * ON-DUTY
   * MEDICAL LEAVE *(or optional split mode)*

7. Color-coded attendance states (clear and minimal)

8. Attendance summary per subject/slot

9. User-defined attendance thresholds *(optional)*

10. Transparent slots (exclude specific slots from attendance stats)

11. Add or remove specific dates or date ranges

12. Automatic save of all data

13. Resume from last viewed position

14. Dark and Light theme support

15. Export attendance data (table/image)

---

## How to Use

### 1. First Launch

* Open the app
* You will see an empty screen
* Follow the on-screen instruction to create a timetable

---

### 2. Create a Timetable

**Steps:**

1. Select the days you attend

2. For each selected day:

   * Create time slots
   * Enter:

     * Start time
     * End time
     * Slot name *(max 20 characters)*
   * Ensure slots do not overlap

3. Select timetable duration:

   * Start date
   * End date

4. Name your timetable *(max 20 characters)*

5. Save

**Notes:**

* Timetables cannot overlap in dates
* Multiple timetables are allowed

---

### 3. View Timetable

* All created timetables appear on the home screen
* They are arranged automatically by date

---

### 4. Mark Attendance

* Tap any slot
* Select status
* Saved instantly

---

### 5. Customize (Optional Settings)

You can configure:

* Attendance thresholds *(define your own ranges)*
* Status mode:

  * Simple → MEDICAL LEAVE
  * Split → MEDICAL SUBMITTED / MEDICAL UPLOADED
* Transparent slots *(exclude from stats)*
* Add or remove dates

All settings are optional. The app works without modifying them.

---

### 6. View Attendance Stats

* Open stats section
* See attendance summary per slot/subject

**Notes:**

* Slots marked as transparent do not appear in stats
* If all entries are “No Class”, attendance is 0%

---

### 7. Share Attendance

* Export your attendance data
* Available as structured data or visual format
